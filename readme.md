# Chess Engine

## Objective:

the goal of this project is to build a chess engine, the project will be divided into 4 phases:

1. Creating the Chess game itself (CLI-based).
2. Adding a web interface (playable on localhost).
3. Adding a computer opponent/bot to play against.
4. Using an AI optimized bot instead.

## Building from Source.

Make sure that cargo and rust are installed on the system.

```
git clone https://github.com/officialar33b/chess_engine.git &&\
cd chess_engine &&\
cargo run
```
